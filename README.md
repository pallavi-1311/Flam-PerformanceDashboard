# Flam Performance Dashboard

A **high-performance real-time data visualization dashboard** built using **Next.js 14 (App Router)** and **TypeScript**.  
This dashboard efficiently renders **10,000+ data points at 60 FPS**, with support for **real-time updates, data aggregation, and performance monitoring**.
